# Verbal Currency — Truth Over Tactics

Multi-page sales & communication mastery website.

## Quick Start

```bash
npm install
npm run dev
```

## Images — Drop into public/images/

```
poster.png       → Movie poster (two men in suits)
stage.png        → Two men on stage with crowd
window.png       → Man at window, cityscape
spotlight.png    → Man on stool, spotlight
filmscape.png    → Two men arms crossed
vikhyat.jpg      → Vikhyat photo (black suit)
```

## Pages

- `/` — Home
- `/about` — Story, values, built-by
- `/faq` — Categorized FAQ
- `/contact` — Application form + Calendly

## Build & Deploy

```bash
npm run build    # outputs to dist/
```

## Backend (Contact Form)

Edit `src/pages/Contact.tsx` handleSubmit to connect to your API or use Formspree/Web3Forms.

## Stack

React 18 · TypeScript · Vite · Tailwind · Headless UI · React Router
